python解压缩学习

python解压缩学习

平常我下游戏或者动漫啥的都是压缩包，还好多都是压缩包嵌套，一般一下也有好几个，还都带着密码，所以我想直接用的脚本全部解压。

7z格式压缩包

这里学习的

[]: https://www.jianshu.com/p/ac643b0c6001

我用的py7zr库

pip install py7zr

解压

import py7zr

with py7zr.SevenZipFile('123.7z', mode='r') as z:
    z.extractall()

创建

with py7zr.SevenZipFile('123.7z', 'w') as z:
    z.writeall('./base_dir')

解压（带密码）

import py7zr

with py7zr.SevenZipFile('123.7z', mode='r', password='secret') as z:
    z.extractall()

创建（带密码）

import py7zr

with py7zr.SevenZipFile('123.7z', 'w', password='secret') as archive:
    archive.writeall('/path/to/base_dir', 'base')

zip格式压缩包

要用zipfile库，python自带的库

解压

import zipfile

zip_file = zipfile.ZipFile(path)
zip_list = zip_file.namelist() # 得到压缩包里所有文件

for f in zip_list:
    zip_file.extract(f, folder_abs) # 循环解压文件到指定目录

zip_file.close() # 关闭文件，必须有，释放内存

解压（带密码）

import zipfile

zf = zipfile.ZipFile("D:/test/t.zip")
zf.extractall(path=".",pwd = "123".encode('utf-8'))

我的脚本

首先是os得到文件，因为有好多是嵌套的，所以我在解压完要再看解压出的文件是否还有压缩包，需要再次解压，这里就涉及到读取文件夹的名字

import os

dirlist = []
filelist = []
filenames = os.listdir(path)
for f in filenames:
    if os.path.isdir(path+'/'+f):
        dirlist.append(f)  #文件夹的列
    elif os.path.isfile(path+'/'+f):
        filelist.append(f)   #文件的列

脚本：

import os
import py7zr

path = "D:\\game"
filenames = os.listdir(path)
passwd = input('密码：')
dirlist = []

for filename in filenames:
    if '.7z' in filename:
        print("try %s" % filename)
        with py7zr.SevenZipFile(filename,mode='r',password=passwd) as z:
            z.extractall()
    elif '.rar' in filename:
        print("try %s" % filename)
        rf = rarfile.RarFile(filename)
        rf.extractall(path=".",pwd = passwd.encode('utf-8'))
    elif '.zip' in filename:
        print("try %s" % filename)
        zf = zipfile.ZipFile(filename)
        zf.extractall(path=".",pwd = passwd.encode('utf-8'))
    print("finish %s" % filename)
for f in filenames:
    if os.path.isdir(path+'/'+f):
        dirlist.append(f)
    # elif os.path.isfile(path+'/'+f):
    #     filelist.append(f)
for f in dirlist:
    filenames = os.listdir(path+'/'+f)
    for filename in filenames:
        if '.7z' in filename:
            print("try %s again" % filename)
            with py7zr.SevenZipFile(path+'/'+f+'/'+filename,mode='r',password=passwd) as z:
                z.extractall()
        elif '.rar' in filename:
            print("try %s again" % filename)
            rf = rarfile.RarFile(path+'/'+f+'/'+filename)
            rf.extractall(path=".",pwd = passwd.encode('utf-8'))
        elif '.zip' in filename:
            print("try %s again" % filename)
            zf = zipfile.ZipFile(path+'/'+f+'/'+filename)
            zf.extractall(path=".",pwd = passwd.encode('utf-8'))
    print("finish %s again" % filename)
print("已全部完成")

最多只能对嵌套了两次压缩包起作用。且只能是解压完后的新压缩包在一个新的文件夹内。
