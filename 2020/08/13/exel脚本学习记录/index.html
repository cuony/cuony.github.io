exel脚本学习记录

今天被安排工具人，汇总这学期所有人得旷到情况，包括次数，和日期，汇总到一个exel表里，给了我10张表，简直不当人了。这自己干就是沙比，就像镭哥所说过的：“现在自己干活的都是傻逼”。开始写脚本，没接触过exel库现学就完事了。
主要就是xlrd和xlwt这两个库。xlwt只能写入.xls格式的

读取：

因为要提取所有旷到的名字还有日期，好在有固定格式，

x人旷到（xxx，xxx，xxx），表的名字里包含日期，信息学院3月18日主题团日活动开展情况.xlsx

我用正则匹配匹配出相应的部分

import re

reg1 = r'信息学院(.*?)主题团日'
reg2 = r'到\（(.*?)\）'

但恶心的地方是用英式中式括号的都有，还有逗号也是英式中式都有，所以我又用replace()函数，把所有的括号和逗号都换成中式的，并把匹配的名字用中式逗号作为分隔符写入列表中

names = names.replace('(','（')
        names = names.replace(')','）')

然后是写入：

记住一点

workbook = xlwt.Workbook(encoding = 'utf-8')
worksheet = workbook.add_sheet('My Worksheet')

这个别放在循环里，要不然每次循环都把exel重置了。

因为前面的name是表，所以我先遍历name，将所有信息都先写入了一张exel。

然后再按顺序读取名字，都先将信息写入字典内，再拿名字依序对照后面的名字，如果有相同的次数就加一，时间就字符串拼接在后面，再写入新表内。

写了一下午，边学边写，主要中间也有段时间思路错了，想开始读取表的时候就一步到位，直接把次数求出来，但太难了。放弃了，就想着先全写入个表里在自己手动统计，好家伙，1000多个了，放弃了，继续写脚本，反而因为有了这个中间表有了新的思路。

import xlrd
from openpyxl import load_workbook
import openpyxl
import os
import re
import xlwt

path = "D:\\作业\\组织部"
result = "D:\\作业\\1.xls"

filenames = os.listdir(path)
reg1 = r'信息学院(.*?)主题团日'
reg2 = r'到\（(.*?)\）'
reg1_list = re.compile(reg1)
reg2_list = re.compile(reg2)
m=0
workbook = xlwt.Workbook(encoding = 'utf-8')
worksheet = workbook.add_sheet('My Worksheet')

for filename in filenames:
    print("try %s" % filename)
    data = re.findall(reg1_list,filename)
    position = path+'/'+filename
    f = xlrd.open_workbook(position)
    sheet = f.sheet_by_index(0)
    for i in range(sheet.nrows):
        names = sheet.cell_value(i,6)
        classes = sheet.cell_value(i,1)
        names = names.replace('(','（')
        names = names.replace(')','）')
        name1 = re.findall(reg2_list,names)
        for i in name1:
            i = i.replace(',','，')
            name = i.split("，")
            for n in name:
                worksheet.write(m,1,classes)
                worksheet.write(m,2,n)
                worksheet.write(m,3,1)
                worksheet.write(m,4,data)
                m=m+1
workbook.save(result)
print("录入结束")

补：

过了几天突然一想，想到这个代码还是有问题，最后得到的并没有把重复的名字去掉，只是把日期加到了后面，且有好多重名情况，所以判断条件再加个同班级。

import xlrd
from openpyxl import load_workbook
import openpyxl
import os
import re
import xlwt

path = "D:\\作业\\组织部"
result = "D:\\作业\\1.xls"

filenames = os.listdir(path)
reg1 = r'信息学院(.*?)主题团日'
reg2 = r'到\（(.*?)\）'
reg1_list = re.compile(reg1)
reg2_list = re.compile(reg2)
m=0
workbook = xlwt.Workbook(encoding = 'utf-8')
worksheet = workbook.add_sheet('My Worksheet')

for filename in filenames:
    print("try %s" % filename)
    data = re.findall(reg1_list,filename)
    position = path+'/'+filename
    f = xlrd.open_workbook(position)
    sheet = f.sheet_by_index(0)
    for i in range(sheet.nrows):
        names = sheet.cell_value(i,6)
        classes = sheet.cell_value(i,1)
        names = names.replace('(','（')
        names = names.replace(')','）')
        name1 = re.findall(reg2_list,names)
        for i in name1:
            i = i.replace(',','，')
            name = i.split("，")
            for n in name:
                worksheet.write(m,1,classes)
                worksheet.write(m,2,n)
                worksheet.write(m,3,1)
                worksheet.write(m,4,data)
                m=m+1
workbook.save(result)
print("录入结束")

workbook = xlwt.Workbook(encoding = 'utf-8')
worksheet = workbook.add_sheet('My Work',cell_overwrite_ok=True)
m=0
aa = {'name':'','num':0,'data':''}
f = xlrd.open_workbook(result)
sheet = f.sheet_by_index(0)
for i in range(sheet.nrows):
    aa['class'] = sheet.cell_value(i,1)
    aa['name'] = sheet.cell_value(i,2)
    aa['num'] = 0
    aa['data']=''
    for i in range(sheet.nrows):
        if aa['name']==sheet.cell_value(i,2) and aa['class']==sheet.cell_value(i,1):
            aa['num'] = aa['num']+1
            aa['data'] = aa['data']+sheet.cell_value(i,4)+'，'
    worksheet.write(m,1,aa['class'])
    worksheet.write(m,2,aa['name'])
    worksheet.write(m,3,aa['num'])
    worksheet.write(m,4,aa['data'])
    m=m+1
workbook.save("D:\\作业\\2.xls")
print("整理完成")

workbook = xlwt.Workbook(encoding = 'utf-8')
worksheet = workbook.add_sheet('My aWork',cell_overwrite_ok=True)
f = xlrd.open_workbook("D:\\作业\\2.xls")
sheet = f.sheet_by_index(0)
for i in range(sheet.nrows):
    aa['class'] = sheet.cell_value(i,1)
    aa['num'] = sheet.cell_value(i,3)
    aa['data']= sheet.cell_value(i,4)
    aa['name'] = sheet.cell_value(i,2)
    worksheet.write(i,1,aa['class'])
    worksheet.write(i,2,aa['name'])
    worksheet.write(i,3,aa['num'])
    worksheet.write(i,4,aa['data'])
    for k in range(sheet.nrows):
        if aa['name'] == sheet.cell_value(k,2) and k != i and aa['class']==sheet.cell_value(k,1):
            worksheet.write(k,1,'')
            worksheet.write(k,2,'')
            worksheet.write(k,3,'')
            worksheet.write(k,4,'')
workbook.save("D:\\作业\\3.xls")
print("删除完成")

这里把相同名字的行数据全部改成空的，重复的就剩最后一个名字那一行，删除完排个序就行了。
